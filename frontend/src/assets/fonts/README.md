# JMH Typewriter font

The "Commentation" header uses **JMH Typewriter** by Jorge Morán.

1. Download it from [DaFont – JMH Typewriter](https://www.dafont.com/jmh-typewriter.font) (free for personal use).
2. Save `JMH Typewriter.ttf` into this folder: `frontend/src/assets/fonts/JMHTypewriter.ttf`.

The build will bundle the font. If the file is missing, the header falls back to system monospace.
