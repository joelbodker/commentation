# JMH Typewriter font

The "Commentation" title uses **JMH Typewriter** by Jorge Morán.

1. Download it from [DaFont – JMH Typewriter](https://www.dafont.com/jmh-typewriter.font) (free for personal use).
2. Save `JMH Typewriter.ttf` into this folder as: `JMHTypewriter.ttf`.

The build will bundle the font. If the file is missing, the title falls back to system monospace (`Courier New`).
